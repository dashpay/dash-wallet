Quelques mots sur la sécurité de vos Darkcoins:

  • Les Darkcoins sont stockés sur l'appareil avec Darkcoin Wallet. Si vous le perdez, vous perdez vos Darkcoins.
  • Cela signifie que vous avez besoin de sauvegarder vos clés privées! Conservez-les dans un endroit sécurisé et souvenez-vous de votre mot de passe.
  • Avant de désinstaller Darkcoin Wallet (ou d'effacer les données de l'application/effacer les données de votre appareil), transférez vos Darkcoins dans un autre porte-monnaie. Tout les Darkcoins restant seront perdus.
  • Les paiements sont irréversibles. Si vous envoyez vos Darkcoins dans le vide, il n'y a aucun moyen de les récupérer sauf si vous connaissez le destinataire et qu'il est d'accord pour vous les renvoyer.
  • Gardez votre appareil mobile en sécurité! N'installez que des applications de confiance. Des applications malveillantes pourraient essayer de voler votre porte-monnaie.
  • Minimisez les risques! Utilisez Darkcoin Wallet seulement pour des petits montants pour un usage quotidien.
